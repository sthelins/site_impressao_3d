# 🖨️ Impressão 3D SM — Site Oficial

Site responsivo da **Impressão 3D SM** — loja de peças únicas impressas em 3D.  
Decoração, miniaturas, utilitários e presentes personalizados.

🔗 **Site:** [impressao3d-sm.github.io](https://impressao3d-sm.github.io)  
📸 **Instagram:** [@impressao3d_sm](https://www.instagram.com/impressao3d_sm)  
📧 **Email:** impressao3d_sm@hotmail.com

---

## ✨ Funcionalidades

- Design **dark & premium** com detalhes dourados
- 100% **responsivo** (mobile, tablet e desktop)
- **Vitrine de produtos** com filtro por categoria
- Seção **Como Pedir** passo a passo
- **FAQ** com accordion interativo
- Botão flutuante de **WhatsApp**
- Animações suaves de entrada
- Zero dependências externas — HTML/CSS/JS puro

## 📁 Estrutura

```
/
├── index.html       ← página principal
└── imagens/         ← fotos dos produtos e da equipe
    ├── foto-equipe.jpg
    ├── produto-1.jpg
    └── ...
```

## 🚀 Como atualizar

1. Edite o `index.html` localmente
2. Adicione fotos na pasta `imagens/`
3. Commit + push → GitHub Pages atualiza automaticamente
